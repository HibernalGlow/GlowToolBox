# GlowToolBox

一个强大的工具箱项目，提供多种实用工具和功能。

## 项目结构

```
GlowToolBox/
├── packages/          # 核心包
│   └── core/         # 核心功能模块
├── apps/             # 应用程序
│   └── tui/         # TUI应用
├── tools/            # 工具和脚本
├── tests/            # 测试文件
├── config/           # 配置文件
├── docs/             # 文档
└── examples/         # 示例代码
```

## 安装

1. 创建虚拟环境:
```bash
python -m venv venv
```

2. 激活虚拟环境:
```bash
# Windows
.\venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. 安装依赖:
```bash
pip install -r requirements.txt
```
